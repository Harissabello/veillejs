JavaScript

Qu'est-ce-que JavaScript

JavaScript est langage de programmation,aussi un langage Orienté Objet. On l'utilise à l'intérieure des pages web. On dit que JavaScript est une extension du HTML.

Hitorique JavaScript

Le JavaScript, inventé par un certain Brendan Eich et développé par Netscape, fait son apparition en 1995, sous le nom de LiveScript, dans le but de dynamiser les pages web. Son utilisation s'est largement répandue, et il se fait rapidement accepter par d'autres navigateurs. Il est aujourd'hui très présent sur les sites web : de plus en plus de webmasters s'y intéressent, et il est de mieux en mieux accepté, à la fois par les navigateurs et par les visiteurs.

Le role de JavaScript

JavaScript dans une page web est de faire apparaître, bouger ou disparaître certains élémentsde la page commeuntitre, un paragraphe, un menu, ou encore une image. l'on peut se servir de ce langage de programmation pour mettre à jour certains èléments de notre page web sans pour autant avoir la recharger. c'est également ce langage qui permet à la page de rèagit quand l'utilisateur clique, fait bouger la souris ou tape sur son clavier.

Les caractéristiques de JavaScript

JavaScripte est un script, un langage orienté objet. Le code n'est pas compilé : il est donc plus rapide à produire (pas besoin de compilateur, un seul fichier, ...), mais il est moins puissant qu'un programme en C, par exemple, et relativement limité : il se limite plus ou moins à la page web sur laquelle il se trouve. Il ne permet donc pas de faire des choses comme manipuler des fichiers sur votre disque dur (heureusement ) ; seulement des choses assez simples. Il est exécuté par le navigateur du visiteur (le client).

A quoi JavaScript sert?

  • Rendre nos pages HTML beaucoup plus interactives
  • Ouvrir des pop-up sur une page web(les petites fenêtres qui s'ouvrent de manière intempestive)
  • Faire défiler un texte insérer
  • Faire un menu dynamique (qui se développe au passage de la souris)
  • Proposer un diaporama (changement d'image toute les X secondes, boutons pour mettre en pause, aller à l'image précédente / suivante, etc.)
